# Week 2 ML Projects – IntelliCrew

## Project Title
 Model Evaluation, Classification & Machine Leaning Implementation

## Objective
This project includes the implementation of Linear Regression from Scratch and a Classification Model to understand Machine Learning concepts, model implementation, evaluation, and GitHub collaboration.

## Dataset

### Linear Regression
A simple synthetic dataset was created manually using NumPy arrays.

**Input (X):**
- [1, 2, 3, 4, 5, 6]

**Output (Y):**
- [2, 4, 5, 4, 5, 7]

The dataset is used to calculate the slope, intercept, predictions, Mean Squared Error (MSE), and visualize the regression line.

### Classification Model
A synthetic customer purchase dataset was generated using NumPy with **1,000 records**.

**Features:**
- Age
- Salary
- Time_Spent
- Pages_Visited
- Previous_Purchases

**Target Variable:**
- Purchased (0 = No, 1 = Yes)

The dataset is used to train a Logistic Regression model for predicting whether a customer will purchase a product.
